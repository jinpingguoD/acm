### 1. 一阶微分方程的求解

这是整个章节的基石，主要考察你对方程形式的观察和变形能力。常见的题型包括：

- **可分离变量的微分方程**
    
    - **形式**：能化为 $g(y)dy = f(x)dx$ 的形式。
        
    - **解法**：直接两边积分，即 $\int g(y)dy = \int f(x)dx + C$。
        
- **齐次微分方程**
    
    - **形式**：能化为 $\frac{dy}{dx} = \phi(\frac{y}{x})$ 的形式。
        
    - **解法**：作变量代换令 $u = \frac{y}{x}$（即 $y = ux$），将其转化为可分离变量的方程。
        
- **一阶线性微分方程**
    
    - **形式**：$\frac{dy}{dx} + P(x)y = Q(x)$。
        
    - **解法**：直接套用通解公式（常数变易法或积分因子法），公式为：
        
        $$y = e^{-\int P(x)dx} \left( \int Q(x)e^{\int P(x)dx}dx + C \right)$$
        
- **伯努利方程 (Bernoulli Equation)**
    
    - **形式**：$\frac{dy}{dx} + P(x)y = Q(x)y^n$ （其中 $n \neq 0, 1$）。
        
    - **解法**：除以 $y^n$，并作变量代换 $z = y^{1-n}$，将其化为一阶线性微分方程求解。
        

### 2. 可降阶的高阶微分方程

这类题目通常涉及二阶或三阶导数，但具有某些缺失的变量，可以通过代换降低阶数求解。

- **$y^{(n)} = f(x)$ 型**
    
    - **解法**：连续进行 $n$ 次积分即可。
        
- **缺 $y$ 型**：$y'' = f(x, y')$
    
    - **解法**：令 $p = y'$，则 $y'' = p'$，原方程降阶为关于 $p$ 和 $x$ 的一阶方程 $p' = f(x, p)$。
        
- **缺 $x$ 型**：$y'' = f(y, y')$
    
    - **解法**：令 $p = y'$，利用链式法则将二阶导写为 $y'' = \frac{dp}{dx} = \frac{dp}{dy}\cdot \frac{dy}{dx} = p\frac{dp}{dy}$，原方程降阶为关于 $p$ 和 $y$ 的一阶方程 $p\frac{dp}{dy} = f(y, p)$。
        

### 3. 高阶常系数线性微分方程

这是考试的绝对重点，通常以二阶为主，解题高度程式化。

- **二阶常系数齐次线性微分方程**：$y'' + py' + qy = 0$
    
    - **解法**：写出特征方程 $r^2 + pr + q = 0$。根据其根的情况（两个不等的实根、两个相等的实根、一对共轭复根），直接写出对应的通解结构。
        
- **二阶常系数非齐次线性微分方程**：$y'' + py' + qy = f(x)$
    
    - **解法**：通解 $=$ 齐次方程的通解 $+$ 非齐次方程的特解（即 $y = Y + y^*$）。通常 $f(x)$ 会是多项式、指数函数、三角函数或它们的乘积，需要使用**待定系数法**来设出特解 $y^*$ 并代入原方程求出系数。
