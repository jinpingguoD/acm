### 1. 矩阵的基本运算

这是矩阵章节的“加减乘除”，是所有后续操作的基础。

- **核心知识点：**
    
    - 矩阵的加法、减法与数乘。
        
    - **矩阵的乘法**（重点：不满足交换律，即 $AB \neq BA$；满足结合律和分配律）。
        
    - 矩阵的转置（Transpose），性质如 $(AB)^T = B^T A^T$。
        
    - 方阵的幂运算与方阵的迹（Trace）。
        
    - 特殊矩阵：对称矩阵 ($A^T = A$)、反对称矩阵 ($A^T = -A$)、正交矩阵 ($A^T A = E$)。
        
- **可能出现的题目：**
    
    - **计算题：** 给出矩阵 $A$ 和 $B$，求 $AB$、$A^2 - 2A + E$ 等。
        
    - **证明题：** 已知某条件，证明某矩阵是对称矩阵或反对称矩阵（通常利用转置的性质来证）。
        
    - **辨析题：** 考察 $(A+B)^2 = A^2 + 2AB + B^2$ 成立的条件（必须是 $AB=BA$）。
        
矩阵乘法，若 A 是 m×n 矩阵， B 是 n×p 矩阵，则 AB 有意义。
乘积 C=AB 将是一个 m×p 的矩阵。
以及关于乘法交换律：A* B=B* A，在矩阵乘法中通常不成立
不过结合律与分配律可以正常使用

### 2. 逆矩阵 (Inverse Matrix)

逆矩阵相当于实数运算中的“倒数”，是解矩阵方程的关键。

- **核心知识点：**
    
    - 逆矩阵的定义：如果 $AB = BA = E$，则 $A$ 可逆且 $A^{-1} = B$。
        
    - 方阵可逆的充要条件：行列式 $|A| \neq 0$（即非奇异矩阵）。
        
    - 伴随矩阵 ($A^*$ ) 与逆矩阵的关系：$A^{-1} = \frac{1}{|A|}A^*$。
        
    - 逆矩阵的性质，如 $(AB)^{-1} = B^{-1}A^{-1}$，$(A^T)^{-1} = (A^{-1})^T$。
        
- **可能出现的题目：**
    
    - **计算题：** 用伴随矩阵法或初等变换法求具体矩阵 $A$ 的逆矩阵 $A^{-1}$。
        
    - **解矩阵方程：** 已知 $AX = B$ 或 $XA = B$，求未知矩阵 $X$（通常通过两边同乘 $A^{-1}$ 求解，即 $X = A^{-1}B$ 或 $X = BA^{-1}$）。
        
    - **证明题：** 证明某个由 $A$、$B$ 组合而成的矩阵是可逆的，并求其逆。
        
只有方阵（行数等于列数）才能形成一个逆矩阵
方法：初等行变换法（将原矩阵与单位矩阵进行拼接，然后运用初等行变换（某行乘非零数、两行互换、某行的倍数加到另一行）进行计算）


### 3. 初等变换与初等矩阵

这是线性代数中最实用、最强大的计算工具。

- **核心知识点：**
    
    - 三种初等行/列变换（交换两行、某行乘非零常数、某行加到另一行上）。
        
    - 行阶梯形矩阵 (Row Echelon Form) 与行最简形矩阵。
        
    - 初等矩阵：对单位矩阵 $E$ 进行一次初等变换得到的矩阵。
        
    - **核心定理：** 初等行变换相当于在矩阵**左乘**相应的初等矩阵；初等列变换相当于在**右乘**。
        
- **可能出现的题目：**
    
    - **操作题：** 将给定矩阵化为行最简形矩阵（这是解线性方程组的基础步骤）。
        
    - **求逆矩阵：** 构造增广矩阵 $(A | E)$，通过初等行变换将其化为 $(E | A^{-1})$。
        
    - **理论题：** 将一个可逆矩阵分解为多个初等矩阵的乘积。
        

### 4. 矩阵的秩 (Rank of a Matrix)

秩是衡量矩阵“有效信息量”或“非冗余维度”的核心指标。

- **核心知识点：**
    
    - 秩的定义：矩阵中非零子式的最高阶数，记作 $r(A)$ 或 $R(A)$。
        
    - 行秩等于列秩。
        
    - 利用初等变换求矩阵的秩（初等变换不改变矩阵的秩）。
        
    - 秩的性质与不等式：如 $r(A+B) \le r(A) + r(B)$，以及 Sylvester 不等式 $r(AB) \ge r(A) + r(B) - n$。
        
- **可能出现的题目：**
    
    - **计算题：** 求给定数字矩阵的秩。
        
    - **含参讨论题：** 矩阵中含有未知参数 $k$，讨论当 $k$ 取不同值时，矩阵的秩分别是多少（通常结合行列式或阶梯形矩阵来做）。
        
    - **证明题：** 利用秩的不等式性质，证明某些矩阵的秩的关系。
        
矩阵的秩简单来说就是一个矩阵的行阶梯形矩阵中的非零行的个数，行阶梯形矩阵必须满足以下三个特征：
1. **零行在下**：如果有全为0的行，它们必须都在矩阵的最下方。
2. **台阶状**：每一行的第一个非零元素（称为主元），必须在上一行主元的**右边**。
3. **下方为零**：每个主元所在列的下方元素必须全为0。

**形象理解**：就像下楼梯一样，非零元素集中在左上角，右下角全是0。
在实际运算中通常可以直接从第一列开始，逐渐向下进行操作
最高阶非零子式：当求最高阶非零子式的时候，首先对其进行化简到行阶梯形矩阵然后看秩的数目
最高阶非零子式通常都是秩* 秩的矩阵，那么找到非零行的主元所在的列数然后外加上前秩行（原矩阵中找出）构成的矩阵一般来说都成立（行列式值不为0），如果不成立那么就找后面的列外加上前秩行（或者无线性相关）
或者说最高阶非零子式就是在这个矩阵中找到相对应的r* r（秩）的一个行列式，且这个行列式值不为0
### 5. 分块矩阵 (Block Matrices)

当矩阵阶数过大时，分块可以极大简化计算。

- **核心知识点：**
    
    - 分块矩阵的加法、乘法（必须满足维数匹配）。
        
    - 分块对角矩阵的幂运算与求逆运算。
        
- **可能出现的题目：**
    
    - **计算题：** 求高阶（如 4 阶以上）矩阵的逆或 $n$ 次方，通常这个矩阵有大量的 $0$，可以划分成分块对角或分块上三角矩阵来简化运算。
