适用于静态数组的查询，但不适合修改，初始化时间复杂度能到NlogN，查询复杂度O（1）
简单来说还是用到了倍增思想，首先对于一个数组来说，第一层先存储这个数组的本身值
然后第二层 三层按照2的幂一次向上增加，最后实现简单的查询
代码：
```cpp
//查询函数
int F1(int l,int r) {
	int k = log_2[r - l + 1];
	//在这里log_2是一个首先需要初始化的值
	//因为是倍增的查询,因此需要先给出最大的查询范围
	//查找左边l对应的值与右边r相对应的值
	return max(st[k][l], st[k][r - (1 << k) + 1]);
}
//首先第一层是数组本身的值
for (int i = 1; i <= n; i++) {
	cin >> st[0][i];
}
//对log_2数组进行初始化
log_2[1] = 0;
for (int i = 2; i <= n; i++) {
	log_2[i] = log_2[i / 2] + 1;
}
//然后就是ST表的构建
//i最大取到n,也就是找到不大于n的最大2次幂,这是层数
for (int i = 1; (1 << i) <= n; i++) {
	//j加上当前的数目不超过n,也就是例如一个层中间存储的是4个数的最大值
	//那么这一层的数量就相当于在原有的数目上-3
	for (int j = 1; j + (1 << i) - 1 <= n; j++) {
		//进行状态转移
		//也就是通过对上一层的数进行倍增,2次幂
		st[i][j] = max(st[i - 1][j], st[i - 1][j + (1 << (i - 1))]);
	}
}
```
