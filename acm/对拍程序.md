首先保证一共有两个程序A B,A是自己写的但不保证对的,B是暴力等方法保证一定对的,那么这种情况下首先将这两个程序放进同一个文件夹中,然后就可以开始写数据代码
```cpp
#include <iostream>
#include <random>
#include <chrono>
using namespace std;
int main(){
	mt19937_64 rng(chrono::steady_clock::now().time_since_epoch().count());
	uniform_int_distribution<int> dist_N(1,20);
	//其中括号里面是范围,但因为只检验对错,可以不同那么大
	uniform_int_distribution<int> dist_K(-100, 100);
	int n=dist_N(rng), k=dist_K(rng);
	//对其随机化值
	cout<<n<<'\n';
	for(int i=0;i<n;i++){
		int a=dist_K(rng);
		int b=dist_K(rng);
		cout<<a<<' '<<b<<'\n';
	}
}
```
类似于上文这种的代码能狗很好的示例，然后就可以再写一个对拍程序就可以比较
```cpp
#include <iostream>
#include <cstdlib>
using namespace std;
int main(){
	int t=0;
	while(1){
		cout<<"test:"<<t<<'\n';
		t++;
		system("shuju.exe > data.in");
		//随机造出来的数据存储在"data.in"中
		system("A.exe < data.in > A.out");
		//直接让A输出的结果存储在A.out中,B同理
		system("B.exe < data.in > B.out");
		if(system("fc A.out B.out > nul"))
		//检验是否相等,如果忽略空格与换行差异可以这样写
		//"fc /W std.out test.out > nul"
		//nul表示忽略怎么比较的,不加可以更直观感受最后WA的地方在哪里,加不加都行
		{
			cout<<"WA!"<<'\n';
			break;
		}
		cout<<"AC!"<<'\n';
	}
	cin.get();
	return 0;
}
```
然后直接点击已经编译过的程序，让其自己在后台跑数据就行，如果遇见WA的情况直接去找data.in里面去找数据就行
